# Power BI and n8n Dashboard

This repository contains data and configurations for a Power BI and n8n workflow project.

## Contents
- **Datasets**: Various CSV files containing location-specific and aggregated data (e.g., Honda datasets).
- **Power BI Dashboard**: `PowerBIDashboard.pbix` containing the visualizations and data model.
- **n8n Workflow**: `My-n8n-workflow.json` containing the automation workflow setup.

## Usage
1. Open the `.pbix` file in Power BI Desktop to view or edit the dashboard.
2. Import the `.json` file into your n8n instance to review or run the automation workflows.
